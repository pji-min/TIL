# 변수 명명 규칙



## 규칙

------

1. **숫자**로 시작할 수 없다 ( 숫자 포함은가능 )
2. 공백 🙅‍♀️
3. 자바 예약어 쓰면 ❌ ( `public` `class` `int` )
4. 영문자( A,B,C ) 달러기호 ( $ ) 밑줄 ( _ ) 만 가능합니다

## Camel case 🐫

------

1. 소문자로 시작한다
2. 소문자로 시작하는 것이 일반적이고, 이후 각 단어는 대문자로 시작합니다 ex) `studentNumber`
3. 하지만!!! `class`는 대문자로 시작, 나머지는 소문자로 시작

하지만 **예외**가 딱 **두개** 있습니다 !!!

1. 상수는 모두 대문자를 사용하고 언더바로 구분해요! ex) `USER_LIMIT`
2. 패키지는 전부 소문자를 사용합니다 ex) `org.spring.boot`

**변수 이름은 의미 있고 그 용도를 명확하게 설명해야 합니다**
