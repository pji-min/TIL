HTTP란 Hyper Text Transfer Protocol 의 약자

- Hyper Text
    
    하이퍼링크를 통해 사용자가 다른 문서로 이동할 수 있는 것
    
- Transfer Protocol
    
    네트워크 상에서 데이터를 어떻게 주고 받을 지 정해 놓은 통신 규약
    

---

### 특징

- TCP/IP 위에서 동작, 기본 포트는 80번
- Stateless(무상태)여서 각 요청은 독립적으로 처리
- 단방향 통신이라 클라이언트 요청 시 서버에서 응답, 서버에서 먼저 응답하지 않음
- 경우에 따라 UDP 사용도 가능

---

### 문제점

- 평문 통신이라 도청 가능
- 상대 확인 불가로 위장 가능
- HTTP는 따로 암호화 과정을 거치지 않기 때문에 중간에 패킷을 가로챌 수 있고, 수정이 가능해서 보안에 취약
