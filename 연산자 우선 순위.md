# 연산자 우선순위

1. 괄호 ()
2. 단항 연산자 ex) `++` `--` `!` `~` `new` `(type)`
3. 산술 연산자 ex) `*, /, %` 다음 `+,-`
4. Shift 연산자 ex) `<<` `>>` `>>>`
5. 비교 연산자 ex) `<` `<=` `>=` `instanceof`
6. 등식 연산자 ex) `==` `!=`
7. 비트 연산자 ex) `&` `^` `|`
8. 논리 연산자 ex) `&&` `||`
9. 삼항 연산자 ex) `? :`
10. 대입 연산자 ex) `=` `+=` `-=` `*=` `/=` `%=`
